# hexo-pupa-theme

个人使用的博客主题

[在线访问](https://xwartz.github.com/pupa)

## 预览

![](http://ww1.sinaimg.cn/large/801b780agw1f7jt2qsaosj210i14mtbf.jpg)

## 安装

```bash
npm install
```

各种依赖参考 [pupa](https://github.com/xwartz/pupa)

## 启用

修改 `_config.yml` 的 `theme` 配置项为 `pupa`:

```yaml
theme: pupa
```

## 更新

``` bash
cd themes/pupa
git pull
```
